## About wcm.io DevOps

wcm.io DevOps is an Open Source project which provides tools for configuration management and deployment.


### Modules

* [CONGA - CONfiguration GenerAtor](conga/)
    * [CONGA Plugin for Apache Sling](conga/plugins/sling)
